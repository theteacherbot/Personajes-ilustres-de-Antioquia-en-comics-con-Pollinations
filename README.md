Una aplicación web interactiva que presenta personajes históricos destacados del departamento de Antioquia, Colombia, mediante una galería de imágenes en formato cómic con transiciones animadas.

## ✨ Características

- 🖼️ **Presentación de diapositivas** con imágenes de personajes ilustres antioqueños
- 🎨 **Transiciones animadas** aleatorias (fade, slide, rotate, scale)
- ⌨️ **Navegación intuitiva** mediante botones, teclas de flecha o indicadores
- 📱 **Diseño responsive** adaptado a diferentes dispositivos
- 🌙 **Interfaz dark mode** con estética moderna
- ⚡ **Carga única** - todas las imágenes embebidas como data URIs

## 🚀 Uso

1. Abre el archivo `Personajes_ilustres_de_Antioquia.html` en cualquier navegador moderno
2. Navega entre las diapositivas usando:
   - Botones ◀ ▶ en los laterales
   - Teclas de flecha ← →
   - Indicadores circulares inferiores

## 🛠️ Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Flexbox, animaciones keyframes, variables CSS
- **JavaScript ES6+** - Módulos, manejo de eventos, DOM dinámico
- **WebP** - Formato de imagen optimizado embebido en base64
