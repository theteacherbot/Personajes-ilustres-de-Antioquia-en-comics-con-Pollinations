# 🎭 Personajes Ilustres de Antioquia en Cómics

Una aplicación web interactiva que presenta personajes históricos destacados del departamento de Antioquia, Colombia, a través de una experiencia de cómic visual.

## 📋 Descripción

Esta aplicación es una presentación de diapositivas interactiva que rinde homenaje a los personajes ilustres de Antioquia, mostrando sus historias y logros mediante imágenes de cómic. La aplicación está diseñada para ser visualmente atractiva y fácil de navegar.

## ✨ Características

- **🖼️ Presentación de diapositivas**: Navegación fluida entre imágenes de personajes históricos
- **🎨 Transiciones animadas**: Efectos visuales aleatorios (fade, slide, rotate, scale) entre diapositivas
- **⌨️ Controles múltiples**: Navegación mediante botones, teclado (flechas) e indicadores táctiles
- **📱 Diseño responsive**: Adaptable a diferentes tamaños de pantalla
- **🌐 Aplicación web progresiva**: Compatible con dispositivos móviles y escritorio

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura semántica y elementos multimedia
- **CSS3** - Animaciones, transiciones y diseño responsive
- **JavaScript (ES6+)** - Lógica de navegación y generación dinámica de contenido
- **Imágenes WebP** - Formato optimizado para carga rápida
